Parse the main formats to json

### How to install

#### For Node.js via NPM
``` bash
npm install --save all2json
```

#### For Node.js via Yarn
``` bash
yarn add all2json
```

### How to use

#### ESModule

``` javascript
import * as All2JSON from 'all2json';
```

#### CommonJS

``` javascript
const All2JSON = require('all2json');
```
